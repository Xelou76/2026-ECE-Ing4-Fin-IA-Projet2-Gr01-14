# Projet IA Finance - ECE Paris 2026

## Groupe 01 Amine Amerdal : Sujet A.5 - Conformal Prediction pour le Risk Management

Ce dépôt contient l'implémentation d'un système de gestion des risques basé sur la Conformal Prediction
appliquée à l'indice S&P 500. L'objectif est de quantifier l'incertitude des prédictions de prix pour sécuriser les décisions d'investissement.
